# Healthcare Data Analytics & Enterprise ETL Project

Description:
This project demonstrates end-to-end data analytics and enterprise ETL
development using SQL, PL/SQL, and Python. It simulates real-world
transactional and analytical workflows including data validation,
automation, reporting, and performance-optimized queries.

Technologies Used:
- SQL, PL/SQL
- Python (pandas, NumPy)
- ETL Pipelines
- Data Validation & Error Handling
- GitHub Version Control

Key Features:
- Enterprise-style PL/SQL packages, procedures, functions, and triggers
- Automated ETL workflows with validation and logging
- Advanced SQL analytics and reporting queries
- Python-based data cleaning and reporting automation
- Scalable and interview-ready project structure
