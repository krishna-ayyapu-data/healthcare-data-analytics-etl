# Healthcare Data Analytics & ETL Project

Description:
This project demonstrates end-to-end data analytics using healthcare datasets.
It includes data cleaning, validation, ETL processing, and dashboard creation.

Technologies Used:
- SQL, PL/SQL
- Python (pandas, NumPy)
- Power BI, Tableau
- ETL & Data Validation
- GitHub

Key Features:
- Automated data cleaning and validation
- SQL-based ETL pipelines
- Performance-optimized queries
- Dashboard-ready analytics outputs
